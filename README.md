# Queensland-IXS
Queensland, Australia, Information Exchange Service

The Information Exchange Service Secure is a cloud based and external managed services 
using XchangeCore/UICDS Web Service Data Orchestration.  
It is deployed in a Clustered environment, with a multi-master MySQL database cluster. 
JASIG Central Authentication Service manages Authentication into all IXP Services.  
To XchangeCore has been added Service Oriented/Representational State Transfer (REST) style web services.  
In the IXS, Request/Incident specific information and business requirements are implemented by an individual IXS (i.e. SES & QDMA).  
Every IXS provides an identical set of external interfaces (REST URL).  
Through XchangeCore, exchange of information is managed on a ‘publish and subscribe’ model.
